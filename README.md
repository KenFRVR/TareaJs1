# Algunas aclaraciones
En el caso de los archivos, los creé con extensión .mjs para que permitiera la importación y exportación de los módulos
En cuanto a las tareas pendientes, asumí que para que una tarea esté pendiente se necesitan 2 factores: que la tarea no se haya realizado y que la fecha de vencimiento
aún no haya llegado, por ese motivo creé el atributo ¨terminada¨
